# Home Lab Network Setup

A simple home lab simulation built in Cisco Packet Tracer demonstrating basic networking setup.

## Tools Used
- Cisco Packet Tracer
- Routers, Switches, and End Devices

## Configuration
- Router: 192.168.1.1
- PC1: 192.168.1.2
- PC2: 192.168.1.3

## How to Run
1. Open `HomeLabNetwork.pkt` in Cisco Packet Tracer.
2. Open PC1 → Command Prompt → type `ping 192.168.1.3`.
3. Network should successfully respond.
